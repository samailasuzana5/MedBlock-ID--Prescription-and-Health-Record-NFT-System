# MedBlock-ID: Prescription and Health Record NFT System
A blockchain-based solution for secure medical records and prescription management.

## 🎯 Features

- Non-transferable NFT-based health records
- Verified doctor and pharmacy registration
- Secure prescription issuance and filling
- Cross-provider record sharing
- Role-based access control

## 🛠️ Technical Stack

- Clarity Smart Contracts
- Stacks Blockchain
- React.js Frontend (recommended)

## 🚀 Getting Started

1. Install Clarinet
```bash
curl -L https://github.com/hirosystems/clarinet/releases/download/v1.0.0/clarinet-linux-x64.tar.gz | tar xz
```

2. Initialize project
```bash
clarinet new medblock-id
```

3. Run tests
```bash
clarinet test
```
